# Christopher Farah

A personal site built with Vike, React, Tailwind, and Bun.

## Install

```bash
bun install
```

## Scripts

```bash
bun run dev
bun run build
bun run lint
bun run lint:fix
bun run preview
```

`sync:package-lock` is available if npm metadata needs to be refreshed:

```bash
bun run sync:package-lock
```
